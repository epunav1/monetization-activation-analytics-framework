# Monetization & Activation Analytics Framework

End-to-end analytics engineering project demonstrating scalable transformation layers, dimensional modeling, and KPI-ready data marts for monetization and measurement workflows.

## Tech Stack
- SQL
- dbt-style transformations
- Dimensional modeling
- Data quality testing

## Architecture
Raw data → Staging → Core (Facts & Dimensions) → Analytics Marts

## Business Use Cases
- Customer activation & retention analysis
- Monetization performance by product & category
- Repeat purchase behavior
- Basket growth and revenue drivers

## Key Outputs
- Activation rate
- Repeat purchase rate
- Basket size & expansion
- Category performance metrics

## Data Quality
- Primary key uniqueness
- Null checks
- Referential integrity
- Duplicate detection

## Future Enhancements
- Incremental models
- Performance optimization strategies
- BI dashboard layer
